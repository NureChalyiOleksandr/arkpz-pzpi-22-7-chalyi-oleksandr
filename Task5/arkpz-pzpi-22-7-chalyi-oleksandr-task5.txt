Хід роботи:
Посилання на відеозапис: https://youtu.be/_dKDrDuI4mI 
Для коректної роботи серверної частини проєкту необхідно встановити .NET Core версії не нижче 8.0. Щоб завантажити репозиторій із кодом проєкту, потрібно виконати команду: git clone "посилання на репозиторій". У проєкті вже налаштовані всі актуальні параметри, тому додаткових змін не потрібно.
Для створення бази даних, яка використовує MSSQL і міграції, слід встановити Microsoft SQL Server Management Studio (SSMS), відкрити проєкт і виконати команду Update-Database.
У проєкті як IoT-пристрій використовується мікроконтролер ESP32, який зчитує дані з датчиків руху та освітлення, а потім передає їх на сервер. Сервер, у свою чергу, обчислює оптимальний рівень яскравості ліхтаря на основі отриманих даних і надсилає його назад на пристрій.
